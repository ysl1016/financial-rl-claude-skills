## 삭제된 파일 상세

### 1. test_claude_integration_full.py
**용도**: Claude API 전체 통합 테스트
**삭제 사유**:
- 초기 개발 단계에서 사용
- 현재는 `tests/integration/test_integration.py`에 통합됨
- 더 이상 필요 없음

### 2. test_claude_simple.py
**용도**: Claude API 기본 연결 테스트
**삭제 사유**:
- API 키 설정 확인용 간단한 테스트
- 일회성 검증용
- 프로덕션에서 불필요

### 3. test_model_versions.py
**용도**: Claude 모델 버전 확인
**삭제 사유**:
- 모델 마이그레이션 시 사용 (claude-3-5 → claude-3-7)
- 마이그레이션 완료 후 불필요
- 일회성 스크립트

### 4. test_spy_data.py
**용도**: SPY 데이터 다운로드 테스트
**삭제 사유**:
- 데이터 처리 테스트는 `tests/unit/test_data_processor.py`에 있음
- 중복 기능
- 임시 검증용

### 5. test_spy_data_no_api.py
**용도**: API 없이 데이터 처리 테스트
**삭제 사유**:
- Claude API 없는 환경 테스트
- 현재는 API 통합 완료
- 더 이상 필요 없음

### 6. test_trading_env_src.py
**용도**: 트레이딩 환경 테스트 (src/tests/에서 복사)
**삭제 사유**:
- `test_trading_env.py`와 동일한 기능
- 파일 이동 시 생긴 중복
- 중복 제거
